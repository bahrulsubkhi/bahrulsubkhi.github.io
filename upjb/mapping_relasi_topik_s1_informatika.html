Pembagian topik — koordinasi antar pengampu
Topik / Algoritma Pengantar KB Kecerdasan Buatan Machine Learning Sistem Cerdas Data Mining Pengolahan Citra
PEMBEDA UTAMA — agar tidak tumpang tindih
1
Pengantar KB vs Kecerdasan Buatan: KB = "kenalan dulu" (konsep & teori tanpa ML), KecBu = "langsung implementasi dengan library." Jangan ajarkan Scikit-learn di Pengantar KB.
2
Kecerdasan Buatan vs Machine Learning: KecBu pakai ML sebagai alat (black box), ML mempelajari cara kerjanya dari sisi matematika. Gradient descent dan backprop masuk ML, bukan KecBu.
3
Machine Learning vs Sistem Cerdas: ML = statistical learning (optimasi fungsi loss), Sistem Cerdas = teknik AI non-statistik (fuzzy, genetika, rule-based) + AI modern (Transformer, XAI).
4
Data Mining vs Machine Learning: DM = deskriptif (temukan pola yang ada), ML = prediktif (bangun model untuk data baru). K-Means di DM = eksplorasi cluster; K-Means di ML = latihan algoritma unsupervised.
5
Pengolahan Citra vs AI lainnya: PCD = matematika citra (filter, transformasi) yang bisa berdiri sendiri. CNN adalah salah satu alat, bukan seluruh matakuliah. Deep learning untuk citra masuk Sistem Cerdas atau MK pilihan.

Pembagian topik — koordinasi antar pengampu
Topik Sis. Digital Org. Arkom Sis. Operasi Sis. Basis Data Jaringan Sis. Terdistribusi Manj. DB Keamanan Info
PEMBEDA UTAMA
1
Sistem Digital vs Org. Arkom: Digital = cara bit diproses (gerbang logika), Arkom = cara bit diorganisir menjadi CPU yang bermakna (instruksi, pipeline, cache).
2
Sistem Operasi vs Arkom: Arkom = hardware yang dikelola, SO = software yang mengelolanya. Proses dan thread masuk SO, bukan Arkom.
3
Sistem Basis Data vs Manajemen DB: SBD = merancang & menggunakan (SQL, normalisasi, ER), MDB = mengelola yang sudah ada di production (index, backup, tuning).
4
Jaringan vs Sistem Terdistribusi: Jaringan = infrastruktur komunikasi (protokol, routing), Sistem Terdistribusi = bagaimana membangun aplikasi di atas banyak node menggunakan infrastruktur itu.

Pembagian topik — tiap MK matematika mengisi celah yang berbeda
Topik / Konsep Kalkulus Aljabar Linear Statistik & Prob. Mat. Diskrit Metode Numerik Teori Bahasa
PEMBEDA UTAMA
1
Kalkulus vs Aljabar Linear: Kalkulus = perubahan pada satu (atau beberapa) variabel kontinu, Aljabar Linear = struktur dan transformasi ruang multi-dimensi. Di Deep Learning: kalkulus untuk backprop, aljabar linear untuk forward pass.
2
Statistik vs tiga lainnya: Statistik satu-satunya yang berurusan dengan data empiris dan ketidakpastian. Tiga lainnya berurusan dengan struktur matematis yang deterministik.
3
Matematika Diskrit vs Kalkulus/Aljabar Linear/Statistik: MatDis = objek terhitung (bilangan bulat, graph, himpunan), tiga lainnya = objek kontinu. MatDis adalah fondasi berpikir algoritmik dan logika, bukan numerik.
4
Metode Numerik vs Kalkulus: Kalkulus = solusi analitik eksak, Metode Numerik = aproksimasi komputasional ketika solusi eksak tidak ada atau tidak praktis. MN adalah "kalkulus yang dijalankan di komputer."

PEMBEDA UTAMA — batas topik antar MK pemrograman
1
Dasar Pemrograman vs Pemrograman Lanjut: Dasar = prosedural murni (variabel, loop, fungsi). Lanjut = rekursi, exception, file I/O, dan pengantar class. Jangan ajarkan inheritance penuh di Pemrograman Lanjut.
2
Pemrograman Lanjut vs PBO: Lanjut = kenalan OOP (class & object), PBO = OOP penuh (inheritance, polymorphism, SOLID, design patterns). Duplikasi di sini paling sering terjadi — koordinasi penting.
3
Pemrograman Web vs Mobile: Sama-sama front-end + back-end, tapi platform dan paradigma berbeda. Web = HTTP/REST/browser, Mobile = lifecycle, sensor, push notification. Backend bisa shared (REST API).
